# SereneMind · Mental Wellness Hub

A calming, feature-rich web application designed to help users manage stress and anxiety through guided breathing, meditation, mood tracking, and daily affirmations.

## ✨ Features

- **4-7-8 Breathing Exercise** — Interactive guided breathing with visual timer
- **Meditation Timer** — Preset sessions (5, 10, 15, 20 min) with start/pause controls
- **Daily Affirmations** — Random positive messages with a single click
- **Mood Tracker** — Log your feelings with visual history and statistics
- **Ambient Sound Player** — Simulated calming sounds (Rain, Waves, Forest, Fire)
- **Calming Activities** — Quick mindfulness exercises (Ocean breath, Forest walk, etc.)
- **Resources Section** — Sri Lanka helplines, YouTube channels, and science-backed articles
- **Responsive Design** — Works on desktop, tablet, and mobile devices

## 🛠️ Technologies Used

- HTML5
- CSS3 (Custom properties, Flexbox, Grid, Animations)
- Vanilla JavaScript (ES6)
- Font Awesome Icons
- Google Fonts (Inter, Playfair Display)
